Getting Started

Installation
Prerequisites:

Before you begin, make sure your development environment includes Node and an npm package manager.

  • Node.js and NPM : You can download Node.js from nodejs.org. NPM comes bundled with Node.js.
Install NPM Packages
  • NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
  • First unzip Main-File.zip you will find two folders including policyhut-nextjs (this is your root folder). Open this folder with VS Code (or any other you like). And follow step below..
  • NPM : You must have to run npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
  • NPM : You can open VS Code built in terminal and run npm install
Development server
  • Run npm run dev for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.
Build for Production Ready for live server

Build you application for host on server using below command:
npm run build

  • In your editor terminal Run npm run build to build the project. The build artifacts will be stored in the dist/ directory. Now you can upload the dist/ folder into your live hosting server's public/ directory
